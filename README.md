# vagrant-devops
Vagrant for development/operations

## This Vagrantfile includes the follows provisioning.

*bootstrap.sh:* Common SO utilities (like htop, git, etc)

*lemp.sh:* Nginx, Mysql and PHP

*node.sh:* NVM (nodejs & npm installers)
